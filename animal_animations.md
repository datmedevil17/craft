# Animal Model Animations

This document lists the available animations for the animal models used in the game.

## Chick and Chicken Animations

The `Chick` and `Chicken` models share the same set of animations:

| Animation Name | Description |
|----------------|-------------|
| `Attack`       | Attack or pecking movement |
| `Death`        | Falling over animation |
| `Idle`         | Basic standing animation |
| `Idle_Peck`    | Pecking at the ground while idle |
| `Run`          | Fast movement animation |

> [!NOTE]
> These models do not have a dedicated `Walk` animation; `Run` is used for movement.

## Pig and Sheep Animations

The `Pig` and `Sheep` models have a more extensive set of animations:

| Animation Name | Description |
|----------------|-------------|
| `Death`        | Falling over animation |
| `Headbutt`     | Attack animation |
| `Idle`         | Basic standing animation |
| `Idle_Eating`  | Bending down to eat |
| `Jump_Loop`    | Mid-air pose |
| `Jump_Start`   | Take-off animation |
| `Run`          | Fast movement |
| `Walk`         | Slow movement |
